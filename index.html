• Saved Settings:

    • Time
      Set the session timer for short 30-60 second blocks or longer as desired. Aim for a total of 5-20 minutes for Desensitization and 2-10 minutes for Installation.

    • Object speed
      A faster speed will make your session more intense. Consider increasing the object speed to push through a difficult Target or emotional trigger.

    • Audio tone
      Select your preferred audio tone. A more grating audio tone can help produce a more intense processing experience.

    • Object size
      Increase or decrease the object size to your preference.

    • Background
      Pick a background that you like.

    • Object color
      Pick an object color that contrasts with the background.

    • Object path
      The default is horizontal, this works for most sessions. Experiment with figure 8 and diagonal paths to see if they are more effective for some issues.
